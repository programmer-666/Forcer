# Forcer
A simple cyber security tool for mysql databases with Python3.
## Requirements
- PyMySQL Module
- Sys Module
- Datetime Module
- Time Module
- Threading Module
- NT or Posix 
## Usage
```bash
$ python3 mysqlforcer.py <host> <username> <wordlist>
```
